THIRD-PARTY LICENSES — scramble visualization bundle
====================================================

The file `bundle.js` in this directory is a prebuilt, minified JavaScript
bundle used to draw 2D scramble diagrams. It is vendored (committed) rather
than built by Gradle. It contains the following third-party components.

Corresponding source / how the bundle is regenerated (pinned versions and the
exact build command) lives in the project repository under:
    tools/scramble-bundle/
